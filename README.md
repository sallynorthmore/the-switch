# The Switch Web site

Based on [Gatsby v2 Beta](https://www.gatsbyjs.org/blog/2018-06-16-announcing-gatsby-v2-beta-launch/).

Documentation on the [Gatsby + Netlify CMS Starter](./docs/README.md)

## Tools

- CSS Modules
- GatsbyJS (ReactJS)
- Netlify + NetlifyCMS

### Getting Started

```
$ npm i
$ npm run develop
$ npm run build
$ npm run serve
```
